# Customer Base Analysis

## Project Overview
This project analyzes customer purchasing behavior using transactional business data. The objective of the analysis was to understand differences in customer value, purchasing patterns, and long-term customer engagement.

The project applies quantitative analytical methods to evaluate how revenue, profit, and transactions are distributed across the customer base and how customer behavior evolves over time.

## Methods Used
- Decile analysis for customer segmentation
- Profit decomposition across customer groups
- Transaction decomposition
- Cohort analysis of customer behavior over time
- Comparison of different customer cohorts

## Tools Used
- Microsoft Excel
- Data analysis techniques
- Business analytics methods

## Key Findings
The analysis shows that customer value is highly concentrated among a relatively small group of customers. The top 20% of customers generate more than half of total revenue and profit. Cohort analysis also shows that customer activity tends to decline over time, indicating opportunities for improving customer retention strategies.
