# Investment Decision Analysis

## Project Description

This project is part of the **Quantitative and Digital Skills Portfolio**. The objective of the project is to analyze investment performance using financial market data and statistical methods. The analysis examines the relationship between the returns of **Kaspi.kz (KSPI)**, the **Freedom 100 Emerging Markets ETF (FRDM)**, and the **S&P 500 market index**.

Historical price data were downloaded from **Yahoo Finance** and analyzed using Microsoft Excel.

## Methods Used

The following financial and statistical indicators were calculated:

* Average Return
* Variance
* Standard Deviation
* Sharpe Ratio
* Covariance
* Correlation
* Linear Regression Analysis

Regression analysis was conducted to evaluate how the returns of the selected assets relate to movements in the broader market represented by the S&P 500 index.

## Key Findings

The analysis shows that both KSPI and FRDM experienced slightly negative average returns during the observed period. Kaspi.kz demonstrates a moderate relationship with the S&P 500 index, while the FRDM ETF shows a stronger positive relationship with the market index. The results highlight differences in volatility, risk-adjusted returns, and market sensitivity between the analyzed assets.
